Windows 7 (64位) 系统下配置Apache 2.4 + PHP 5.5 + MySQL

安装 PHP 5.5

  • 下载安装包，选择 VC11 x64 Thread Safe （VC11线程安全版。Thread Safe为安全线程，与Apache搭配环境；Non Thread Safe为非安全线程，主要与IIS搭配环境）

  • 将下载的压缩包解压到D:/php， 复制该目录下的php.ini-production ，将复制后的文件重命名为php.ini，打开文件修改如下：

    查找：extension_dir = "./"，修改：extension_dir = “D:\php\ext”，配置扩展库的路径；

    查找：;extension=PHP_MySQL.dll，去掉前面的分号：extension=PHP_MySQL.dll；

    查找：;extension=PHP_mysqli.dll，去掉前面的分号：extension=PHP_mysqli.dll；

安装Apache 2.4

  • 下载：选择 httpd-2.4.12-win64-VC11.zip ， 解压安装包后得到Apache24文件夹， 将文件夹拷贝至D盘根目录D:/下（以D盘根目录为安装目录）

  • 修改默认路径：打开 D:\Apache24\conf\httpd.conf， 查找替换所有的默认路径C:/为D:/

  • 安装Apache：以管理员权限打开cmd命令行，依次运行命令：

    D:
cd Apache24\bin
httpd.exe -k install
httpd.exe -k start|stop|restart

  • Apache服务开启|重启|关闭

    net start|restart|stop Apache2.4

配置Apache加载PHP

  • 打开Apached的安装目录，D:\Apache24\conf\extra，新建一个名为httpd-php5.conf的文本文件，打开添加以下内容： 
    LoadModule php5module "D:\php\php5apache22.dll"
AddType application/x-httpd-php .php
AddType application/x-httpd-php-source .phps
DirectoryIndex index.php
PHPIniDir "D:\php"
    打开D:\Apache24\conf\httpd.conf，在文件末尾添加一行： 
    Include conf/extra/httpd-php5.conf
  • 重启Apache服务，打开目录Apache24\htdocs，删除文件：index.html，新建文件：index.php，输入内容： 
     <?php phpinfo(); ?>
    在浏览器地址栏输入localhost，如果显示了PHP信息则配置成功。
